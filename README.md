# Healthcare-data-analysis
An interactive Healthcare Sales Analysis Dashboard designed to provide clear insights into  <br>
sales performance, customer distribution, and regional trends using visual analytics.

# Project Overview -
This dashboard analyzes healthcare sales data across multiple years, quarters, and countries. It helps stakeholders quickly understand order volume, quantity sold, customer trends, and geographic distribution.
